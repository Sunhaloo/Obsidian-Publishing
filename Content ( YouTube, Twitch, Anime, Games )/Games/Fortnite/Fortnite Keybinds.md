# Keybinds

## Movement Keybinds

```console

Forward: W
Backwards: S
Left: A
Right: D
Sprint: Shift
Crouch: Ctrl
Jump / Mantle: Space

```

## Weapon Keybinds

```console

Harvesting Tool: 3
First Slot: Z
Second Slot: Forward Side Mouse Button
Third Slot: C
Fourth Slot: 1
Fifth Slot: 2

```

## Building Keybinds

```console

Wall: Q
Ramp: E
Floor: V
Cone: Back Side Mouse Button
Trap: G
Edit: F

```
